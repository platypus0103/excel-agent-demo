# 設備成本滾算工具使用說明

# 功能概述

這個工具整合了設備成本滾算服務 (`equipment_cost_services.py`)，可以讓 AGENT 執行以下操作：

1. **讀取 Excel 資料夾中的檔案數據** - 從 `Excel/` 資料夾中的 Excel 檔案讀取專案數據
2. **執行價金滾算計算** - 使用四種不同的滾算模式計算設備成本
3. **計算成本結構調整** - 計算信邦利潤和開發費用的影響
4. **輸出結果到 Excel Final 資料夾** - 將結果寫入 `Excel final/滾算後記錄.xlsx` 檔案

# 支援的滾算模式

# 1. 現金模式 (Cash Mode)
```python
tool.execute_price_rolling(
    mode="cash",
    boundary=20000,      # 價金調整邊界
    step=2000,           # 每次減少的現金金額
    profit_rate=0.1,     # 信邦利潤率 (10%)
    development_fee=2000 # 開發費用
)
```

# 2. 比率模式 (Ratio Mode)
```python
tool.execute_price_rolling(
    mode="ratio",
    boundary=15000,      # 價金調整邊界
    step=0.05,          # 每次減少的比率 (5%)
    profit_rate=0.12,   # 信邦利潤率 (12%)
    development_fee=1500
)
```

# 3. 條件模式 (Conditional Mode)
```python
tool.execute_price_rolling(
    mode="conditional",
    boundary=16000,
    profit_rate=0.15,
    development_fee=2200,
    maximum_value=50000,    # 判斷價金範圍的最大值
    minimum_value=25000,    # 判斷價金範圍的最小值
    condition_step_1=3000,  # 價金 > maximum_value 時的 step
    condition_step_2=1500,  # minimum_value <= 價金 <= maximum_value 時的 step
    condition_step_3=800    # 價金 < minimum_value 時的 step
)
```

# 4. 自訂模式 (Customize Mode)
```python
# 自動配置
tool.execute_price_rolling(
    mode="customize",
    boundary=18000,
    adjustment_times=5,     # 滾算次數
    profit_rate=0.08,
    development_fee=1800,
    auto_config=True        # 自動配置 steps
)

# 手動配置
tool.execute_price_rolling(
    mode="customize",
    boundary=18000,
    profit_rate=0.08,
    development_fee=1800,
    auto_config=False,
    custom_steps=[2000, 1500, 1200, 1000, 800]  # 手動指定每次的 step
)
```

# 輸入數據結構

工具會從 `Excel/` 資料夾中的 Excel 檔案讀取以下數據：
- 專案名稱
- 裝置容量 (kWp)
- 起始/結束年度
- 年發電量
- 電價相關參數
- 設備成本（作為滾算的初始價金）
- 貸款利率
- 開發費用
- 維護、保險、回收成本等

# 輸出結果

結果會寫入 `Excel final/滾算後記錄.xlsx` 檔案，包含：
- 計算模式
- 初始價金
- 邊界值
- 調整步幅
- 滾算次數
- 滾算結果列表
- 成本結構調整後的值
- 信邦利潤記錄

# 在 AGENT 中使用

此工具已註冊到 `ToolManager` 中，AGENT 可以透過以下方式調用：

```python
from tool.tool_manager import ToolManager

# 創建工具管理器
tm = ToolManager()

# 執行價金滾算
result = tm.execute_tool(
    tool_name="execute_price_rolling",
    arguments={
        "mode": "cash",
        "boundary": 20000,
        "step": 1500,
        "profit_rate": 0.1,
        "development_fee": 2000
    }
)

if result["success"]:
    print(f"滾算完成，結果已寫入: {result['output_file']}")
    print(f"滾算摘要: {result['summary']}")
else:
    print(f"滾算失敗: {result['message']}")
```
